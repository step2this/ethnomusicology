Implementation Notes
  • Add tabindex="0" to all <tr> rows. Enter/Space selects row, ↑/↓ navigates between rows.
  • Slide-over panel: trap focus with Tab cycling within panel contents. ESC closes and returns focus to triggering row.
  • Add visually-hidden skip link before icon rail: <a href="#track-table" class="sr-only sr-only-focusable">Skip to track catalog</a>
  • Focus ring: outline: 2px solid #E8963A; outline-offset: 2px on all interactive elements. Visible against #0F0D0B and #141210.
  • Convert track list from <div> grid to semantic <table role="grid"> with <th scope="col"> for each column.
  • Add aria-sort="ascending|descending|none" to sortable <th> elements. Update dynamically on sort change.
  • Wrap sort status in aria-live="polite" off-screen region. Update text on each sort action.
  • Toast container: add role="status" and aria-live="polite". Screen readers announce on insert.
  • Album art: add role="img" and aria-label="[Track] by [Artist]" to each thumbnail element.
Deferred to Option B/C
  • Energy bars: add numeric text alternative or pattern fill for color-blind users (WCAG 1.4.1)
  • Source badges: increase from 9px or add title tooltip for readability (WCAG 1.4.12)
  • Icon rail: add aria-label to each button and visible tooltip on hover/focus (WCAG 4.1.2)
  • Scrim dismiss: add ESC key handler to close slide-over from scrim click target (WCAG 2.1.1)
