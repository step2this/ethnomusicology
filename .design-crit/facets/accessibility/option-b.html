Additions Beyond Audit — Quality-of-Life Enhancements

What Option B adds over Option A

  • Keyboard shortcut hints — Persistent bar showing navigation keys: arrow, Enter, Space, Esc
  • ARIA landmarks — nav (icon rail), main (track table), complementary (slide-over), status (toast area)
  • Row count announcement — "19 of 347 tracks visible" via aria-live when filter/sort changes
  • Column resize keyboard support — Implementation note for future: allow keyboard-driven column width adjustment
  • Energy text labels — Numeric value (e.g., "5/10") visible on hover/focus, not just tooltip
  • Icon rail visible tooltips — Pill-style tooltip appears to the right on hover/focus, not only aria-label
  • Source badge readability — 10px + 0.5px letter-spacing (up from 9px in the base design)
