Flat Five
Five core fields at co-primary emphasis: Artist, Title, BPM, Key, Energy. No single column dominates — each is differentiated by visual treatment, not size hierarchy. Artist is bold (the "who"), Title is regular weight (the "what"), BPM is monospace (the "speed"), Key is a colored chip (the "harmony"), Energy is a bar graph (the "intensity").
Design principle: Flat hierarchy, rich visual coding. Each column has a distinct visual voice so the DJ's eye learns to jump to whatever matters for the current task.
