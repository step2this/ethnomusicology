Screen Details & States
Import Screen
Source tab selection, URL input, import execution, progress and summary display.
  • DefaultSource tabs visible, URL field empty, Beatport tab selected
  • URL EnteredURL validated (green check or red error), Import button enabled
  • ImportingProgress bar, track count updating, cancel available
  • CompleteSummary card: N tracks imported, N skipped. "View Catalog" CTA
  • ErrorNetwork/auth/invalid URL. Retry button, error description
  • Partial ImportSome tracks succeeded, some failed. Summary + error details
Track Catalog
Enriched track tiles with BPM, Camelot key, genre, source badges. Browse imported library.
  • DefaultTrack tiles loaded, showing all imported tracks
  • Empty (first use)No tracks yet. CTA: "Import your first tracks"
  • LoadingSkeleton tiles while catalog loads
  • ErrorFailed to load catalog. Retry option
