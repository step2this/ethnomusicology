Screen Details & States
Track Catalog Primary
Enriched track list. The DJ's library view. Always the landing screen.
  • DefaultTracks loaded, enriched tiles with BPM/key/genre
  • Empty (first use)Prominent "Import your first tracks" CTA, import sheet auto-opens
  • LoadingSkeleton tiles
  • ErrorFailed to load catalog, retry option
  • Import activeSheet overlay visible, catalog dimmed behind it
Import Sheet Secondary
Overlay with source tabs, URL input, and import progress. Focused task, dismissible.
  • DefaultSource tabs + URL input, Beatport selected
  • URL EnteredValidation feedback, Import button enabled
  • ImportingProgress bar, track count, can't dismiss
  • CompleteSummary, "Done" button to close sheet
