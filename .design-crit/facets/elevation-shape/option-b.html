Wire Lift

Mostly flat, with targeted shadows on overlays and primary actions. Shadows reserved for things that float.

Elevation Scale

4 levels — shadows on levels 2-3 only

0 Base Canvas none Page background, table body
1 Content Surface none Track rows, header bar, detail strip, icon rail
2 Raised 0 1px 3px rgba(0,0,0,0.4) Toasts, primary button, sort dropdown
3 Overlay 0 4px 16px rgba(0,0,0,0.5) Slide-over panel, tooltips

The main catalog table (levels 0-1) stays completely flat — no shadows on rows, headers, or the detail strip. Shadows appear only when something floats above the table: the slide-over panel, toasts, sort dropdowns, and the primary CTA button. This keeps the daily-use view clean while giving depth cues to overlaid elements.
