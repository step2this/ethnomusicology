Flat Crate

Zero shadows. All depth via borders, background tints, and a scrim. DJ software flat aesthetic.

Elevation Scale

3 levels — no box-shadow anywhere

0 Base Canvas none Page background, table background
1 Content Surface none Track rows, header bar, detail strip, icon rail, toasts
2 Overlay Surface none Slide-over panel, sort dropdown, tooltips

Depth is communicated entirely through background tint steps (#111 → #181 → #1E1) and 1px borders. The scrim (55% black) behind the slide-over provides the strongest depth cue. No shadow rendering cost.

Shape Language

4-token radius scale

0px none
table cells, rows
3px xs
Camelot chips, badges
6px sm
buttons, inputs, toasts
full pill
source pills, sort badges

No medium or large radius tokens. The crate is a dense tool — nothing needs 12px+ corners. Rows are square (0px). Chips are barely rounded (3px). Buttons/inputs are subtly softened (6px). Only source pills go full round.
