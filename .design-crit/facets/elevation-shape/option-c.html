Elevation Scale

5 levels — progressive shadow scale

0 Base Canvas none Page background
1 Content Surface 0 1px 2px rgba(0,0,0,0.3) Track rows, icon rail, detail strip
2 Raised 0 1px 4px rgba(0,0,0,0.35) Header bar, buttons, source pills
3 Overlay 0 3px 10px rgba(0,0,0,0.4) Toasts, sort dropdown
4 Critical Overlay 0 6px 20px rgba(0,0,0,0.5) Slide-over panel, tooltips

Every surface above the base canvas casts a shadow. The scale doubles in blur radius at each step (2 → 4 → 10 → 20px). Opacity increases from 0.3 to 0.5. This creates a smooth, perceptible depth gradient from the flattest row to the most prominent overlay.
