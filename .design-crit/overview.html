Ethnomusicology — Import & Catalog

Multi-source import screen and DJ metadata catalog for an LLM-powered DJ setlist platform. Desktop-first, designed for DJs who think in BPM, key, and energy during set prep.

Defining the product
Screen Inventory
locked — Catalog-First
Review
user decided · 1 round

Decision: 1 primary screen (Track Catalog) + 1 secondary (Import Sheet overlay). Catalog is home base, import is a drawer/sheet action. Extensible for multiple sources and AI generation.

User signal: "DJs browse far more than they import. The catalog IS the product."
View
Edge States
locked — Context-Rich
Review
user decided · 1 round

Decision: Tailored edge states per situation. Empty catalog shows source cards. URL errors show valid examples. Errors reassure "your tracks are safe." Partial imports show per-track results.

User signal: "Context-rich approach is the right fit for this product."
View
Arranging the pieces
Layout
locked — Full-Width Crate
Review
user decided · 1 round

Decision: Edge-to-edge track table with sortable columns. 48px icon rail reserves space for future browser pane. No max-width — DJ software density.

User signal: "V1 discipline — ship the dense track table now, expand later."
View
Content Hierarchy
locked — Flat Five
Review
user decided · 1 round

Decision: 5 co-primary fields (Artist, Title, BPM, Key, Energy) differentiated by visual treatment not size. Bold text, regular text, monospace, color chip, bar graph. 3 emphasis levels total. Task-flexible scanning.

User signal: "All five data points matter — artist, title, BPM, key, and energy. Don't pick a winner."
View
Component Design
locked — Crate + Board
Review
user decided · 2 rounds

Decision: 14 components. C's structure (slide-over with scrim, click-to-select accent border, expandable detail strip, multi-column sort with priority numbers, source pills) with A's design language (compact pill Camelot chips, thin vertical energy bars, flat borders, 6px radius, 34px buttons, 320px panel).

Round 1: Three options generated (Lean Crate, Studio Deck, Mixer Board). User cut A and B, kept C's structure but wanted A's compact design language. Requested A+C merge.
Round 2: A+C merge "Crate + Board" presented. Locked immediately.

User signal: "Merge, don't compromise. C's structural patterns are critical. A's compact language keeps it dense. Sortable columns are non-negotiable."
View
Elevation & Shape
locked — Wire Lift
Review
user decided · 1 round

Decision: 4-level elevation. Table is flat (no shadows). Shadows only on floating elements: slide-over (directional), toasts, primary CTA. Scrim + shadow for overlay depth. 4-token radius scale (0/3/6/full).

User signal: "Differences are subtle — trusts designer recommendation on technical facets."
View
Making it feel right
Typography
locked — Inter Tight
Review
user decided · 1 round

Decision: Inter for UI + JetBrains Mono for BPM/data. 7-level discrete scale (15/13/13/12/11/10/9px). 3 weights (400/500/700). Tabular numbers. ~175KB one-time load.

User signal: "Type matters. Clean sans-serif, Helvetica vibes. Chose quality and cross-platform consistency over zero-load pragmatism."
View
Color System
locked — Ember Crate
Review
user decided · 1 round

Decision: Warm dark palette. Amber (#E8963A) primary, copper (#C06030) secondary on warm-tinted blacks (#0F0D0B). Warm white text (#F0ECE4). 24 warm-shifted Camelot key colors. Energy bars: teal→amber→copper-red. Roland TR-808 aesthetic.

User signal: "It's like a Roland 808. Warm amber is the brand. Subtle hardware references are welcome."
View
Density & Spacing
locked — Studio
Review
user decided · 1 round

Decision: 40px rows, ~19 tracks visible in 900px viewport, 4px base unit. 32x32 art thumbnails. 7-token spacing scale (4/8/12/16/24/32/48px). Matches wireframe density from all prior facets.

User signal: "Not fussed about density — trusted designer recommendation. Studio density matches the wireframes already approved."
View
Final check
Accessibility
locked — Enhanced Access
Review
user decided · 1 round

Decision: WCAG AA, 12/12 issues fixed. Keyboard row navigation (arrow keys + Enter/Space), focus trap on slide-over, skip link, 2px amber focus rings, semantic table with aria-sort, live regions, energy bar tooltips, icon rail tooltips, keyboard shortcut hints, row count announcements.

User signal: "DJs actively seek hotkeys — keyboard shortcuts are a feature, not just compliance."
