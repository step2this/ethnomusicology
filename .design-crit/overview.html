Ethnomusicology — Import & Catalog

Multi-source import screen and DJ metadata catalog for an LLM-powered DJ setlist platform. Designing the tabbed import flow (Spotify + Beatport) and enriched track tiles with BPM, Camelot key color chips, genre, and source badges.

Confirmed Brief

Target Users
DJs building sets from digital catalogs. Desktop-first (laptop at desk), during set prep. Think in BPM, key, and energy.
Core Interaction Loop
Source tab → URL → import → enriched track list
Differentiator
DJ-grade metadata at a glance. Beatport native BPM/key. Camelot color-coding for harmonic awareness.
Platform
Flutter Web, desktop-first, responsive to tablet
Tech Stack
Flutter / Dart Material 3 Riverpod GoRouter
